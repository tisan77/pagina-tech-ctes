**Description**: Mobile-First E-commerce Catalog for 'Tech Ctes' optimized for iPhone 16 Pro with a premium, tech-focused UI. Includes product filtering, search, and WhatsApp integration instead of a traditional checkout.
**Files**:
- index.html: Main entry point and layout structure. Refs: TailwindCSS, Lucide Icons, Google Fonts.
- styles.css: Custom styling, typography rules, glassmorphism, and safe area insets.
- data.js: Mock product database.
- app.js: Application logic for rendering, filtering, searching, and WhatsApp URL generation.
